Para instalar el sistema web:

Descarge el repositorio en su carpeta "htdocs" ( o donde en la carpeta donde trabaja con su servidor ).
Cree una base de datos con el nombre "lcc".
Exporte el script "lcc(1).sql" contenido en el repositorio con su gestor de base de datos ("phpmyadmin" por ejemplo).
Ingrese al index del sistema (http://localhost/xxx/xxx/index.php)
